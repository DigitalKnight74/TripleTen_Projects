## 📌 Project Overview

A multi-task ML project for the **Sure Tomorrow Insurance Company**, evaluating how machine learning can solve four distinct business problems — including a data privacy challenge requiring mathematical obfuscation of client personal information.

---

## 🎯 Objectives

### Task 1 — Customer Similarity
Find customers similar to a given customer to support targeted marketing using k-Nearest Neighbors.

### Task 2 — Benefit Prediction (Classification)
Predict whether a new customer is likely to receive an insurance benefit. Compare trained model performance against an untrained dummy baseline.

### Task 3 — Benefit Count Prediction (Regression)
Predict the number of insurance benefits a new customer is likely to receive using linear regression.

### Task 4 — Data Masking & Obfuscation
Develop a data transformation algorithm to protect clients' personal data (data masking) without degrading ML model quality. Prove mathematically that the algorithm works correctly.
