## 📌 Project Overview

Mobile carrier Megaline found that many of their subscribers are still using legacy plans. Built a classification model to analyze subscriber behavior and recommend one of Megaline's newer plans: **Smart** or **Ultra**.

Behavior data was available from subscribers who had already switched to the new plans. The goal was to develop a model with the highest possible accuracy — with a minimum threshold of **0.75**.

---

## 🎯 Objectives

- Build a binary classification model to recommend a mobile plan
- Evaluate and compare multiple classification algorithms
- Achieve an accuracy score of at least 0.75 on the test dataset
- Tune hyperparameters to optimize model performance

## 📊 Key Results

- Evaluated Decision Tree and Random Forest classifiers across varying hyperparameters
- Achieved target accuracy threshold of 0.75 on the test set
- Random Forest outperformed Decision Tree with optimal depth and estimator settings
