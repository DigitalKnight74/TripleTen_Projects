## 📌 Project Overview

Worked for the online store Ice, which sells video games all over the world. Analyzed user and expert reviews, genres, platforms (e.g. Xbox or PlayStation), and historical sales data to identify patterns that determine whether a game succeeds or not.

Using data going back to 2016 — with the scenario set in December 2016 — built a forecast to plan advertising campaigns for 2017.

---

## 🎯 Objectives

- Identify patterns that determine video game success across regions
- Analyze the impact of platform, genre, and review scores on sales
- Apply statistical hypothesis testing to validate findings
- Build data-driven recommendations for advertising campaign planning
